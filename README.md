### 项目描述

https://github.com/liyincode/immortal

一款融合 Web3 和 AI 裁判的知识答题擂台竞技游戏，可依托 Monad 并行告诉写入的特性，观众可通实时为选手加油鼓掌，并且替换连续答错的选手上台对决，增加刺激性，所有比赛结果永久记录在区块链上实现真正的"不朽"。
